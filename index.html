HTML

  • The head element contains information about the webpage, for the browser.
  • Metadata is data about the webpage that the browser uses to effectively serve it to users.
  • The body element represents the visible content shown to the user.
  • h1 to h6 elements represent the level of heading a given text block represents.
  • Elements that are nested inside other elements are called child elements.
  • The element that holds all the children is known as the parent element.

CSS

  • CSS defines elements' size, position, font, and color.
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • CSS file needs to be linked to HTML in head data to apply.
  • Class selectors allow any HTML element assigned to it to gain their property assignments from the CSS.

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git checkout branch-name: switches to an existing branch including main
  • git add -A: stages all changes
  • git commit -m "message": adds changes and commits locally with desciptive message
  • git pull origin main: fetches and merges any commits from the main branch of a GitHub reposition
  • git push origin feature-name: pushes local changes to the remote repository branch

JavaScript

  • Control flow is the order in which a computer executes code in a script from top line to bottom.
  • A variable is a named container that allows us to store data in our code.
  • An array is a single variable that is used to hold a group of related data
  • An index is a unique number identifying the location of a data item in its array.
  • A for Loop uses the predictable pattern of indices to perform a task on all the items in an array.
  • A function is a set of instructions that tells the computer how to perform a certain task.
